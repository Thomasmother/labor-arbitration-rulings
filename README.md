# 劳动仲裁裁决书 — 工资争议专题

> 14 份广州及珠三角地区劳动人事争议仲裁裁决书（已脱敏）

## 选题理由

当前各大裁判文书网及商用法律数据库（中国裁判文书网、北大法宝、聚法案例等）对公开裁判文书的**脱敏工作已相当成熟**，法院判决书在公开前均已完成当事人信息匿名化处理。相比之下，**劳动人事争议仲裁裁决书**的公开程度和脱敏标准尚不统一，部分地区（如广东佛山、深圳）刚刚开始推行仲裁文书网上公开制度，裁决书的脱敏工作仍由各仲裁委自行把握，研究价值更高。

此外，笔者近期代理了一起劳动仲裁案件，对仲裁庭关于**工资流水不连续时平均工资的认定标准**有不同看法。劳动案件具有**很强的地区性特色**——各地仲裁委在举证责任分配、工资标准认定、经济补偿金计算等核心问题上存在较大裁量差异。因此，先行收录广州及珠三角地区的仲裁裁决书，作为后续类案研究的素材基础。

## 数据来源

| 项目 | 说明 |
|------|------|
| **来源机构** | 佛山市劳动人事争议仲裁委员会及各区分会 |
| **公开平台** | 佛山市人力资源和社会保障局官网 — [仲裁文书公开专栏](https://hrss.foshan.gov.cn/ztzl/ldgxytjzc/zcwsgk/) |
| **文书类型** | 劳动报酬、工资差额、经济补偿金等工资争议类裁决书 |
| **时间跨度** | 2022–2026 |
| **原始形态** | PNG 原件扫描（未经任何文字处理，保留全部原始信息） |

## 数据采集方法

使用 **OpenCLI** 浏览器桥接工具（Browser Bridge Extension），通过 Chrome 浏览器直接访问佛山市人社局仲裁文书公开专栏，自动化完成以下流程：

```
搜索页面 → 遍历各区列表页 → 提取裁决书PNG扫描件 → 本地保存
```

OpenCLI 基于真实浏览器会话，可正常访问政府网站并处理页面渲染，避免了传统爬虫方案对 `.gov.cn` 类网站的反爬触发问题。

## 数据处理方法

### 清洗流程

1. **OCR 文字提取**：使用本机已安装的 Tesseract 5.4（chi_sim + eng 语言包），对 PNG 扫描件进行中文文字识别
2. **字间距修复**：OCR 输出的中文字符之间通常存在多余空格（逐字识别导致），编写正则规则合并相邻中文字符，恢复紧凑排版
3. **脱敏处理**：向 Claude AI 发送脱敏指令，结合其他 AI 工具的建议优化提示词，对以下信息进行统一替换：

| 原始信息类型 | 替换为 | 示例 |
|-------------|--------|------|
| 个人姓名 | 姓 + 某 | 罗海波 → 罗某 |
| 有限公司/股份公司 | 某公司 | 招商局仁和人寿保险股份有限公司 → 某公司 |
| 个体工商户 | 某个体工商户 | — |
| 其他组织（银行/学校/律所等） | 某机构 / 某中学 | 佛山市第三中学 → 某中学 |
| 身份证号码 | XXXXXXXXXXXXXXXXXX | — |
| 统一社会信用代码 | XXXXXXXXXXXXXXXXXX | — |

4. **文件名脱敏**：同步替换文件名中的公司名称和个人姓名
5. **YAML 元数据注入**：为每份裁决书添加结构化 Front Matter，包含案号、来源、OCR引擎、下载日期等字段
